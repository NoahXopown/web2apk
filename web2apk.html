FAQ
Pertanyaan yang sering ditanyakan
Apakah website yang di-deploy permanen?
Vercel free tier menyimpan deployment tanpa batas waktu selama project aktif. URL akan tetap bisa diakses.
Berapa lama proses build APK?
Biasanya 1–3 menit tergantung kompleksitas website dan beban server. Progress ditampilkan secara real-time.
Apakah HTML Extractor bisa scan semua website?
Sebagian besar website bisa di-scan. Website dengan proteksi Cloudflare ketat atau login-wall mungkin tidak bisa diakses.
Di mana API key disimpan?
Semua API key (Vercel tokens, APK apikey) disimpan di file system.js dalam konstanta APK_CONFIG dan VERCEL_TOKENS.

FAQ
Pertanyaan yang sering ditanyakan
Apakah website yang di-deploy permanen?
Vercel free tier menyimpan deployment tanpa batas waktu selama project aktif. URL akan tetap bisa diakses.
Berapa lama proses build APK?
Biasanya 1–3 menit tergantung kompleksitas website dan beban server. Progress ditampilkan secara real-time.
Apakah HTML Extractor bisa scan semua website?
Sebagian besar website bisa di-scan. Website dengan proteksi Cloudflare ketat atau login-wall mungkin tidak bisa diakses.
Di mana API key disimpan?
Semua API key (Vercel tokens, APK apikey) disimpan di file system.js dalam konstanta APK_CONFIG dan VERCEL_TOKENS.
